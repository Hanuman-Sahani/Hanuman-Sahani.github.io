AI / Computer VisionHurema HR & Payroll

AI Face-Recognition Attendance

I engineered a Python gate service that turns IP cameras into attendance sources. The pipeline runs RTSP → detection & tracking → quality gating → ArcFace embeddings → FAISS ANN matching → temporal voting → HMAC-signed punches into the Laravel API. An enrollment pipeline builds the FAISS index from payroll DB face photos, and a live dashboard streams per-camera MJPEG feeds and recognition events over REST. Stack: Python, ArcFace/InsightFace, FAISS, ONNX Runtime, OpenCV, WebSocket, Laravel.

Enterprise PlatformSTM Retaspect

RetaSpect — Retail Field-Force Management

As Team Lead & Project Coordinator I led client requirement meetings, finalized the SOW and managed end-to-end web + mobile delivery. The platform enforces a 5-level hierarchy (Owner → Brand Admin → Regional Manager → Team Leader → Field Staff) with strict RBAC and level-wise data visibility. Flutter field modules include multi-layer attendance (geo-tagging, geo-fencing, selfie face verification, IP validation), store IN/OUT, beat plans with TL approval, campaign execution, sales/pre-booking punching, surveys, targets & incentives, reimbursements and L&D assessments.

Developer ToolPersonal Project

Apex — Offline-First API Client

A cross-platform, Postman-style API client (Windows/macOS/Linux/Android/iOS) built on Clean Architecture + BLoC with get_it DI and functional error handling (Either/dartz); 100% local persistence via SQLite + SharedPreferences. It ships a tabbed workspace, nested drag-and-drop collections, environment variable substitution at send time, rich response viewers, request history, code-snippet generation for ~36 languages and a WebSocket/Socket.IO console. Windows releases are automated via GitHub Actions — portable zip + Inno Setup installer attached to tagged Releases.
